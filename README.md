# dom-reset-css-anim

Reset CSS animation

## Installation

Install with [component(1)](http://component.io):

```bash
$ component install nk-components/dom-reset-css-anim
```

## Usage

```js
var resetCSSAnim = require('dom-reset-css-anim');
var el = document.querySelector('#el');
el = resetCSSAnim(el);
```
